5.11 Select Configuration

This screen displays information about hardware devices chosen for updates, and sets the software that will update those devices.

Depending on the chosen hardware type, several software selections are displayed. Software is only listed if it is applicable to the hardware type selected. The following options may be available.

  • Bridge: Allows the selection of the Bridge software version to update to the device(s). Available Bridge versions are determined by installed environments. In general, use the latest version of this software.
  • SDK: Allows the selection of the SDK software version to update to the device(s). Available SDK versions are determined by installed environments.

The Start button begins the update process.
