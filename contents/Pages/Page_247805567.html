5.8 Environment Destination

Install To Disk

Based on the installation specifications, this option begins the process of installing an environment directly to the development system. 


Install To Archive

This option begins the process of creating a MegaZarf for a new environment. A MegaZarf is a file that contains all the information necessary to recreate that environment in the future. MegaZarf files enable users to consistently and reliably recreate a development environment on any computer that is capable of running NDI. This option does not install any software to the development system.
