4.11 Importing a Development Environment (MegaZarf Format)

Nintendo Dev Interface (NDI) can import a MegaZarf for creating a new environment. Depending on the process used to create the MegaZarf, a connection to the server may be required. By default, MegaZarf files contain all the required data to recreate the environment they contain.

To use a MegaZarf to create an environment, follow these steps.

  1. Start NDI and select Dev Environments.
  2. Click Create From Archive. If there is already an environment on your system, first click New Environment above the environment list on the left of the screen.



  3. In the MegaZarf File field, click the Browse button to select the MegaZarf to be installed.
  4. In the New Environment Location and New Environment Name fields, enter a name and location for the environment being created. Note: Some MegaZarf files prevent changes to either the Location, Name, or both.



  5. Click Install to start the installation process.
  6. After the installation has completed, click Finish.
  7. NDI returns to the Dev Environments list and displays the new environment.
