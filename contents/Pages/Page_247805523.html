5.1 Choose Hardware

This screen lists the Nintendo hardware devices that are accessible by the development system for software updates. The information displayed is as follows.

  • Name: The name given to the device.
  • Type: The platform type of the device.
  • Location: The network address of the device.
  • SDK Version: The current version of the SDK installed to the device. 
  • HostBridge Version: The current version of the HostBridge installed to the device.

A device may be selected for update by clicking on its row in the list. Multiple hardware devices may be selected for a simultaneous update by pressing CTRL and clicking on each desired device. Multiple types of development platforms may be listed, but only devices of a single type may be simultaneously selected and updated.

The Re-scan button reloads the hardware device list.

The Add to List button will search the network for a device with a Location equal to the value entered in the text box. If a device is found with a matching IP address, it will be added to the list.
