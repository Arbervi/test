5.14 Update Hardware

This screen displays the progress of software updates to development hardware devices. A real-time update for each device and file is indicated in a progress bar for each device.

By default, the log window for a device is not displayed. To display the output log of the update for a particular device, click the Show Log button. If multiple devices are being updated, only a single log can be displayed at one time. To close the log for a particular device, click the Hide Log button. A log must be closed to open a different device's log window.

The Cancel button to the right of the progress bar for an individual device, stops the updates in progress for that device.

The Abort button in the lower right of the window, stops the updates in progress for all devices.

Hardware Update Termination

If you terminate a hardware update before the process is completed, your device may enter an inoperable state and any maintenance will require assistance from Nintendo support.
