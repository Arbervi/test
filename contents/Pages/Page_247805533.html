5.4 Configure Environment

This screen allows for selection of the software packages that will be installed when an environment is created. Packages are selected using one of two methods: Selecting sets of software by Bundle or choosing specific software through Detailed Package Management.

Bundles

Bundles are sets of software packages which have been grouped together for a specific purpose. This might be because they are useful to a particular development role, such as Artist or Tester, or because the software has interdependent features. Multiple bundles may be selected at the same time.

The Bundles list is displayed on the left side of the screen. Click a bundle to select or deselect it, which will select or deselect all contained packages for installation. The Installation Summary on the right side of the screen indicates the packages that will be installed with the current selection, including the size of each package and the total download size for the installation.

The Details icon at the right of each selection opens the Bundle Contents for for that bundle, displaying the name and version of all included software packages.

The Select All check box selects or deselects all bundles in the list.

The contents of a bundle may change when multiple bundles are selected to ensure compatibility of all included software.

Detailed Package Management

Selecting the Detailed Package Management link opens the package selection screen.

This screen allows for the direct selection of specific software versions from a list of those available. It is recommended for experienced users. Any packages belonging to bundles that have been selected, will automatically be set to install when leaving the bundle selection screen.

The screen is separated into three sections.

  • Tag List: Located on the left side of the screen, this list contains defined tags which can filter the package list. If there are no tags associated with any available packages, this section is not displayed.
  • Package List: Located in the center of the screen, this list contains details about packages that can be installed.
  • Installation Summary: This list displays all packages that are currently selected to be installed.

Tag List

If any packages in the main package list have tags associated with them, the tag list is displayed. Selecting options from the list filters the package list so that it displays only packages that have the indicated tag. Selecting more than one tag displays only package that have all the selected tags. To remove a selected tag, click the selection again. To show or hide all tags in a particular category (such as Occupation), click the header.

Package List

The package list displays all packages that can be installed or modified, based on the selection of foundation and foundation version for the environment. In addition, it contains several features to assist in package selection.

  • Search Bar: Entering text into the search bar displays packages where the title, description, or tags contain the specified text. To filter, enter the desired text and press ENTER. You may enter text multiple times, in which case the list will display only packages that contain all entered terms. You may remove the last search term by using the DELETE key or remove any search term by clicking the blue "x" icon next to the term.
  • Select All: Set each package, currently in the package list, to install the latest available version.
  • Clear All: Deselect all packages that are currently in the package list.
  • Other Actions
    • Select All Platform Add-Ons: Sets all add-ons for packages in the current package list, to install.

  • Name: The indicated name for the package.
  • Action Selection: A drop-down list box that contains actions that may be performed on the package. Click the button to cycle through all available actions. An icon displayed next to the action indicates what action is being taken. For more information about specific icons, see the Action Icons table below.
    • Not Included: The package will not be installed.
    • Version Selection: The specified version of the package will be installed. Depending on the software, several different versions may be available.
  • Add-On List (If applicable): Add-ons selected from this list are installed for this package. Selected add-ons are added to the Installation Summary. An add-on package requires the package it is included with to be installed, or selected for installation to be selected. 
  • Description: A general description of the package contents. If the description is large, Show More is displayed. Selecting Show More opens a window with the full text of the description.
  • Tags: Tags that the package belongs to (if any) are displayed in the lower left. Selecting a tag behaves in the same way as selecting a tag in the tag list.

Installation Summary List
