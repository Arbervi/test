4.8 Setting a Default Environment

To function correctly, some features of external or development programs require a default path to SDK files. Nintendo Dev Interface (NDI) can quickly assign an environment as the default path for a particular type of hardware, which can be changed when desired. This can be helpful when changing between different projects that use the same software.

To set a default environment, follow these steps.

  1. Start NDI and click Dev Environments.
  2. Select the environment that you want to set as default.
  3. Click the Manage button and select Set as Switch Default from the menu.
