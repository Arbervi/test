4.9 Using a Dashboard

In addition to software, environments may also install a dashboard, which is located on the Dev Environments screen after the environment is selected. The dashboard contains commands specific to the installed software and shortcuts to Nintendo Dev Interface (NDI) tasks, such as initialization of development hardware or installation of additional packages. To add a dashboard item to the Welcome screen of NDI,  click the star icon next to the dashboard item. Optionally, provide the dashboard with a custom name.
