4.1 Creating a Development Environment with Mastering Options

Mastering information determines which environments may be submitted to Nintendo for review. Nintendo Dev Interface (NDI) displays the types of mastering that are available for a foundation version when an environment is created. A date range is listed for each mastering option, determining when submissions of that type will be accepted. NDI also prevents you from updating to a version without mastering information, if a development environment with mastering was installed. 

To create an environment and determine mastering options, follow these steps.

  1. Start NDI and select Dev Environments.
  2. Click Create My Own. If there is already an environment on your system, before clicking Create My Own, click New Environment above the environment list on the left of the screen.
  3. Click Install To Disk.
  4. If not already logged in to NDI, a dialog box is displayed requesting that you to log in to your account. Before continuing further, this step must be completed by using a Nintendo Developer Portal ID.
  5. After completing the log in process, select which Platform to develop for, any additional Tool Set (these are standalone options that can be installed alongside the SDK, such as additional documentation), and the Foundation, which forms the core of your new environment (generally, the core SDK software). After making your selections, click Next to continue.
  6. Select the version of the Foundation software to install. An information icon () located on the left of a version, indicates that it contains mastering information. To view mastering information, click the information icon. 
    • The Master Submission Information dialog displays timing information for each type of submission and any additional mastering notes for the selected version. The listed dates indicate when Nintendo will accept submissions of that type for consideration.



  7. On the choose location screen, enter the directory on the system where the environment will be installed to. Ensure there is enough storage space to hold the current software and to expand in case of future updates. Click Next to continue.
  8. On the configuration screen, in addition to the Foundation software, you can select additional software bundles that contain sets of software to install. After all selections are made, click Next.
  9. A summary of the software to be installed is displayed, including the total file size of the final environment. 
  10. Click Install to start the download and installation process.
  11. After the installation has been completed, click Finish. 
  12. NDI returns to the Dev Environments list and displays the new environment.
