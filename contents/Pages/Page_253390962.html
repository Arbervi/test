4.6 Updating Development Hardware

To run software created on development hardware (such as an SDEV or EDEV), the device must have an appropriate version of the SDK installed. Nintendo Dev Interface (NDI) allows you to install the correct files by selecting the environment that you are working in and the SDK version required to run your software is then installed.

When an appropriate SDK is installed, follow these steps to update hardware.

  1. Start NDI and select Hardware.
  2. From the presented list, click the hardware device you want to update, then click Next. You may select multiple units by pressing CTRL and clicking the units to select them.



  3. Select the environment that you want to be compatible with your hardware from the SDK drop-down list box on the right, then click Start.



  4. Your hardware is updated automatically.
