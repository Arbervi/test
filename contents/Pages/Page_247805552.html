5.6 Define New Environment

This screen selects the hardware platform and (optionally) an additional tool set for an environment, which determines the software packages that can be installed as the environment is created.

Platform

This selection determines the hardware platform for which the environment being installed will be created. It also determines which tool sets and foundations are available for selection.

Tool Set

This selection determines which tool set (if any) will be installed with an environment. A tool set is complementary software which may be installed alongside the chosen foundation, including software documentation or game engines such as Unity or Bezel Engine. To install only the foundation software (such as an SDK), select the Standard option.


Info

When a Documents Tool Set is selected, some environment configuration options are omitted.
