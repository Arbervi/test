5.12 Select SDK Version

This screen determines the version of the foundation that will be installed. 


Masterable Information

Masterable versions of software have been validated for submission to Nintendo for approval. If an add-on or Foundation version is masterable, the mastering details icon () is displayed to the left of the version number. Click the icon to open a dialog box that contains information regarding mastering for that version.

In the dialog, the types of mastering for the software are displayed. A date range next to each type determines when the software may be submitted for that type of mastering. The dates are color coded as indicated in the following list.

  • Green: The software is currently masterable.
  • Yellow: The software is currently masterable, but will no longer be masterable after the second displayed date.
  • Red: The software is not currently masterable. Mastering may have been closed or not yet opened for the software, depending on the displayed dates.
